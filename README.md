# father-dumi-comp-demo

## Getting Started

Install dependencies,

```bash
$ npm i father-dumi-comp-demo
```
or   

```bash
$ yarn add father-dumi-comp-demo
```


Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
